Descendant combinators

These combine two selectors such that the elements matched by the second selector are selected if they have an ancestor (parent, parent's name, parent's parent's parent, etc.) element matching the first selector. They are generally represented by a single space (" ") character.

body article p {
}

Selectors that utilize a descendant combinator are known as descendant selectors.

Combinators

Combinators combine other selectors to target groups of elements:

article > p {
}

Combining selectors and combinators

You can combine multiple selectors and combinators together:

article p span {
color: orange;
}
This would change a <span> located within a <p> inside an <article> to the color orange.

You can also combine multiple types together:

body h1 + p .special {
color: yellow;
background-color: black;
padding: 5px;
}

This will style any element with a class of special, which is inside a <p>, which comes just after 
an <h1>, which is inside a <body>.
