<H1> headings should only be used once per page as they contain information that is referenced for SEO purposes. If you need to replicate the style of an <H1> heading for design purposes, utilize <span> elements to stylize text accordingly. Use <H2>, <H3>, etc. for subheadings, sub-subheadings, etc.

Use <mark> to highlight text

Use <em> to italicize text--don't use <i>

Use <strong> to bold text--don't use <b>

Roots, URLs, and paths

The "root" refers to the name of the folder that houses all of your website's index file, code, directories, documents, images, etc.

  • Index.html files are used for your landing pages. You can have multiple index files, but they have to be located in different directories (sub-folders) within the root. This means that only one index.html file can be present directly inside the root and outside of your directories.
    • See here for information on how to use multiple index files in the same website, etc.
    • Keep in mind that HTML files do not have to be named "index.html"; you can easily create subpages containing HTML that have different file names--just be sure to link to them accordingly (see above link or here for basic idea).

Links

Document fragments are used to jump to a certain section on a given page. This page can be either the current page or a different page altogether. They use the href attribute with a # symbol in <anchor> elements to jump to the section of the page with the corresponding ID attribute. They are case-sensitive.

Example code:

<a href="#doc-frag-ex">click here<a>
  • Be strategic with links--search engines catalog hyperlinked text for SEO purposes.

The download attribute can be used to create a name for files to be saved as when downloaded by another system.

Input:

<a href="https://download.mozilla.org/?product=firefox-latest-ssl&os=win64&lang=en-US"
download="firefox-latest-64bit-installer.exe">
Download Firefox for Windows (64-bit) (English, US)
    </a>

Output:

Download Latest Firefox for Windows (64-bit) (English, US)

Description lists

Note the use of two <dd> elements in the last <dt> element

Input:

<dl>
   <dt>soliloquy</dt>
        <dd>In drama, where a character speaks to themselves, representing their inner 
            thoughts or feelings and in the process relaying them to the audience 
            (but not to other characters.)</dd>
    <dt>monologue</dt>
        <dd>In drama, where a character speaks their thoughts out loud to share them 
            with the audience and any other characters present.</dd>
    <dt>aside</dt>
        <dd>In drama, where a character shares a comment only with the audience for
            humorous or dramatic effect. This is usually a feeling, thought, or piece 
            of additional background information.</dd>
        <dd>In writing, a section of content that is related to the current topic, but 
            doesn't fit directly into the main flow of content so is presented nearby 
            (often in a box off to the side.)</dd>
  </dl>

Output:

soliloquy
In drama, where a character speaks to themselves, representing their inner thoughts or feelings and in the process relaying them to the audience (but not to other characters.)
monologue
In drama, where a character speaks their thoughts out loud to share them with the audience and any other characters present.
aside
In drama, where a character shares a comment only with the audience for humorous or dramatic effect. This is usually a feeling, thought, or piece of additional background information.
In writing, a section of content that is related to the current topic, but doesn't fit directly into the main flow of content so is presented nearby (often in a box off to the side.)
