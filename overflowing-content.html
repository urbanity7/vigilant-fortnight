What is overflow?

Overflow happens when there is too much content to fit in a box. Remember, everything in CSS is a box. Because CSS tries to avoid "data loss," it has a tendency to permit overflow unless otherwise instructed not to.

The overflow property

The overflow property is used to control overflow. It is set to visible by default, which allows overflow to be seen on the frontend. To crop content when it overflows, you can use overflow: hidden. Because this makes overflow invisible, only use this when you're sure hiding content won't cause problems. Finally, using overflow: scroll adds scrollbars when content overflows to keep the layout consistent while showing all pertinent content. To scroll on just one axis instead of both, you can use the corresponding property for that axis (e.g., overflow-y: scroll).
